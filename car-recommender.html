Smart Car Recommender

Personalized Vehicle Filtering System

PythonStreamlit Data AnalysisUI/UX

1. The Problem

Buying a used car can be overwhelming due to the sheer volume of choices. The goal of this project was to create an intuitive interface that allows users to find their ideal vehicle by narrowing down thousands of listings based on specific, practical constraints.

Interactive Filtering Logic

Users can input five critical parameters to find their match:

  • Budget Management: Set a strict maximum price (e.g., $6,000).
  • Usage Threshold: Filter by maximum odometer mileage to ensure reliability.
  • Vehicle Specs: Select by type (Sedan, Truck, SUV) and engine power (Cylinders).
  • Quality Control: Filter results by condition, from "Fair" to "Excellent".
